Description

VIP Membership

Allows admins to move users to certain groups for a certain period of time and put them back to a group of their choice once the expiration date has passed.

Admins are PM'd when someone adds someone to a new membership.

Admins are PM'd when someone's membership expires.

Users whose membership expires, are alerted too. (via PM)

Install

Upload contents of the Upload folder to the root of your MyBB installation.

Go to Admin CP -> Configuration -> Plugins and Activate "VIP Membership"

Then go to Settings -> VIP Membership and change anything you need.

Upgrade

From 1.6 to 1.7:

  • Upload contents of the Upload folder to the root of your MyBB installation, overwriting existing files.
